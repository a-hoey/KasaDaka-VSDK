Welcome to the web interface of La Merveille des Femmes. Here you can find the information that is stored in the database sorted by diagnosis. You can find symptoms that are related to certain diagnoses under the header “symptoms”. Specific cases are stored under the header “cases”.

If you are a physician and you would like to add information to the system, you are asked to call the number 020-369 76 64 and choose the option “add information to the system”. If you have any questions or concerns about the content of this web page, you can send an email to our system owner at systemowner.merveilledesfemmes@gmail.com.

Note that if you are not a physician, this web page can not be used for medical advice on complaints. If you are a pregnant woman or consult this web page for a pregnant woman with specific symptoms during her pregnancy, you are urged to consult a doctor, who may consult this system for more information.
