Bienvenue sur l'interface web de La Merveille des Femmes. Ici vous pouvez trouver les informations stockées dans la base de données triées par diagnostic. Vous pouvez trouver des symptômes liés à certains diagnostics sous les «symptômes» d'en-tête. Des cas spécifiques sont stockés sous l'en-tête "cas".

Si vous êtes médecin et que vous souhaitez ajouter des informations au système, vous devez appeler le numéro 020-369 76 64 et choisir l'option "ajouter des informations au système". Si vous avez des questions ou des préoccupations concernant le contenu de cette page Web, vous pouvez envoyer un courriel à notre propriétaire du système à systemowner.merveilledesfemmes@gmail.com

Notez que si vous n'êtes pas médecin, cette page Web ne peut pas être utilisée pour des conseils médicaux sur les plaintes. Si vous êtes une femme enceinte ou si vous consultez cette page Web pour une femme enceinte présentant des symptômes spécifiques pendant sa grossesse, vous êtes invité à consulter un médecin, qui pourra consulter ce système pour plus d'informations.
